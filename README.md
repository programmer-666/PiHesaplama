<h3>Pi sayısını hesaplamada kullanılan ana formül</h3>
<p>Yazılan program Pi'nin en doğru şekilde hesaplanmasına yöneliktir.Virgülden sonra ki basamaklarını hesaplayamazsınız.</p>
<br>
